Eight new public interfaces — `CallToolHandlerConfig` / `CallToolHandlerCustomizer` (`server.tools`), `GetPromptHandlerConfig` / `GetPromptHandlerCustomizer` (`server.prompts`), `ReadResourceHandlerConfig` / `ReadResourceHandlerCustomizer` and `ReadResourceTemplateHandlerConfig` / `ReadResourceTemplateHandlerCustomizer` (`server.resources`). Each Config exposes `descriptor()` / `method()` / `bean()` getters and an `interceptor(...)` mutator returning the config for chaining. Mutable config impls are package-private static nested classes inside each `*Handlers` helper. The three autoconfigs (`MocapiServerToolsAutoConfiguration`, `MocapiServerPromptsAutoConfiguration`, `MocapiServerResourcesAutoConfiguration`) gained `@Autowired(required = false) List<*HandlerCustomizer>` parameters and pass them through. Existing 5 callers of `*Handlers.build` were updated to pass `List.of()` for customizers.
